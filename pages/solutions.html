Chamely

AI Homework Tutor

A Socratic homework helper that supports students without giving shortcuts. Chamely guides reasoning step-by-step, using textbook-aligned context, adaptive hints, and visual explanations — not answer dumps.

Curriculum fidelity, not hallucination. Responses grounded in actual textbook content via RAG.

Socratic-style guidance that strengthens reasoning and builds genuine understanding.

Built to strengthen teaching, not replace it. Designed with educators in mind.

Key Capabilities

  • OCR homework extraction from photos
  • Socratic reasoning flow with guided questions
  • Adaptive hints: questions, diagrams, scaffolding
  • Gamification: coins, badges, streaks, challenges
  • Curriculum-aligned responses via RAG
  • Topic mastery insights for students
  • Content struggle patterns for publishers
  • Building metacognitive skills
  • Routine question handling so classroom time can focus on deeper learning
